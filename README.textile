h1. 欢迎来到NxOgre中文版wiki

h2. 前言

正如你所看见，这里没有代码(但不代表以后没有)。这里就是为初学nxogre的新手一条捷径。
曾几何时，我也钻研过nxogre，怎奈网上教程太少，英文版教程倒是有，可惜E文不好，大多看不懂。只好放弃
但是随着学习的深入，逐渐对nxogre有了更深一步的了解，并且E文随着编程的深入也不断增强，看懂一些简单的文章或者英文网页
也不是什么难事了。到现在，编程也卓有成效了，回过头想想，如果有完备的教程也不至于走这么多的弯路，因此就推出了本教程，
意在为还在自学编程之路徘徊的新手们一个少走弯路的机会。

h2. 关于本教程

本教程完全是根据betajaen写的nxogre英文原版教程翻译而来，不存在机器翻译。文档格式采用的textfile，部分也采用markdown格式。
个人感觉textfile实在是太强大了，甚至比html都强大，都简便。国外wiki教程用的都是textfile，因此强烈建议大家学会textfile，也方便
之间的交流。
